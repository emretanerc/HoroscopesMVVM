
### HoroscopesMVVM

This application was created based on daily, weekly, monthly and annual interpretations of horoscopes.
There was not much emphasis on design. It is a study on technical details.


- MVVM 
- LiveData
- Hilt
- Retrofit & RxJava3
- UseCases
- Jetpack Navigation
- Picasso Library
- One Signal
- Build SRC
- Light & Dark Mode
- Responsive Design
